Zasady pisania w HTML 5

  1. Nigdy nie krzyżuj znaczników.
  2. Zawsze zamykaj znaczniki.
  3. Zawsze małe litery przy atrybutach.
  4. Używanie cudzysłowia dla atrybutów.
  5. Używanie znaków spacjalnych - encji.
